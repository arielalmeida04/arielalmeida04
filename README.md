## 🚀 Proyecto Destacado  

### 📚 ApiLibros  

API REST desarrollada con **Java y Spring Boot** para la gestión de libros, aplicando buenas prácticas de arquitectura backend.  

### ✨ Funcionalidades  
- ✔️ CRUD completo  
- ✔️ Persistencia con JPA / Hibernate  
- ✔️ Arquitectura en capas (Controller - Service - Repository)  
- ✔️ Manejo de endpoints REST  
- ✔️ Integración con base de datos relacional  

### 🛠️ Tecnologías del proyecto  
- Java  
- Spring Boot  
- Spring Data JPA  
- Hibernate  
- PostgreSQL  

### 🎯 Objetivo del proyecto  
Aplicar conceptos reales de desarrollo backend como diseño de APIs REST, separación de responsabilidades y persistencia de datos.
